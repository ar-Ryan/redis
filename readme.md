### Running the Script in Linux and macOS (Bash)

1. Open a terminal.
2. Navigate to the project directory.
3. Make the script executable (only needed once):
    ```bash 
    chmod +x run.sh
    ```
4. Run the script:
    ```bash
    ./run.sh
    ```
